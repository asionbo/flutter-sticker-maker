# Flutter Sticker Maker Example

This example demonstrates the features of the Flutter Sticker Maker plugin, including:

- Creating stickers by removing image backgrounds
- Customizing border colors and widths
- iOS-style lift animation on long press

## Features Demonstrated

### 1. Background Removal
Pick an image from your gallery or capture one with the camera, then create a sticker by removing the background.

### 2. Border Customization
- Toggle borders on/off
- Choose from preset border colors
- Adjust border width with a slider

### 3. Lift Animation (New!)
Long press on any image (original or sticker) to see the iOS-style lift animation effect:
- Smooth scale animation
- Elevation/shadow effect
- Visual feedback with snackbar notification

## Getting Started

1. Grant necessary permissions (camera and photo library access)
2. Select or capture an image
3. Customize border settings if desired
4. Create your sticker
5. Try the lift animation by long-pressing on the images!
